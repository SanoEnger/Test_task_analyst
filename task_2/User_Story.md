# User Story: Публикация товара на маркетплейсе

**Как** Продавец,
**Я хочу** иметь возможность публиковать информацию о своих товарах в личном кабинете,
**Чтобы** они стали доступны для просмотра и покупки потенциальными клиентами на витрине маркетплейса.

## Критерии приемки (Acceptance Criteria):
1. **Заполнение данных:** Продавец может указать название, описание, категорию, цену и загрузить изображения товара.
2. **Черновики:** Продавец может сохранить заполненные данные как черновик, чтобы вернуться к редактированию позже.
3. **Валидация:** Система не должна позволять публикацию без заполнения обязательных полей (Название, Категория, Цена).
4. **Модерация:** После нажатия кнопки "Опубликовать" товар отправляется на проверку модератором.
5. **Статус публикации:** Продавец видит актуальный статус товара (Черновик, На модерации, Опубликован, Отклонен).
6. **Уведомления:** Продавец получает уведомление (в ЛК или на почту) о результатах модерации.
